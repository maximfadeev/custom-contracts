To generate custom contract enter contract name, token name and pairs of address & percentage. If all is ok, new contract will be printed to console, otherwise will print error. 

`node index.js contract-name token-name`
`node index.js contract-name token-name SP1 10`
`node index.js contract-name token-name SP1 10 SP2 20`
`node index.js contract-name token-name SP1 10 SP2 20 SP3 10`
